# Ultimate Windows System Tool v2.0 🚀

**Complete all-in-one system optimization, cleaning, and maintenance tool for Windows**

## 📋 Overview

This powerful CMD script provides comprehensive Windows system management with 25+ advanced features including cleanup, optimization, repair, security, and monitoring tools - all without requiring additional software installation.

## ✨ Key Features

### 🧹 System Cleanup (Functions 1-4)
- **Quick System Cleanup** - Fast cleanup of temp files, cache, and junk
- **Deep System Cleanup** - Thorough cleaning including Component Store, Windows.old
- **Browser Cache Cleanup** - Cleans Chrome, Firefox, Edge, Opera, Brave caches
- **Application Cache Cleanup** - Removes cache from Discord, Teams, Spotify, VS Code, etc.

### ⚡ System Optimization (Functions 5-8)
- **Performance Boost** - Optimizes services, visual effects, Game Mode, power plan
- **Memory (RAM) Optimization** - Frees RAM, disables SuperFetch, optimizes paging
- **Network Optimization** - Optimizes TCP settings, DNS, disables throttling
- **Disk Optimization** - Defragmentation, TRIM support for SSDs

### 🔧 Repair and Maintenance (Functions 9-12)
- **Automatic Windows Repair** - Runs SFC, DISM to fix system files
- **Fix Windows Update Issues** - Repairs Windows Update components
- **Network Reset and Repair** - Complete network stack reset
- **Disk Check and Repair** - CHKDSK with auto-repair scheduling

### 🛡️ Security and Privacy (Functions 13-16)
- **Privacy Cleanup** - Removes activity traces, history, telemetry data
- **Startup Programs Manager** - Opens Task Manager for easy startup control
- **Malware Scan and Cleanup** - Windows Defender quick scan
- **Security Optimization** - Enables firewall, UAC, SmartScreen

### 📊 System Information (Functions 17-20)
- **Detailed System Information** - Complete hardware and OS details
- **Disk Space Analysis** - Shows disk usage and largest files (100MB+)
- **Running Processes Monitor** - Display top RAM-consuming processes
- **System Health Check** - Comprehensive health diagnostics with battery report

### 🔥 Advanced Tools (Functions 21-25)
- **Create System Restore Point** - Manual restore point creation
- **Delete Windows.old Folder** - Safely remove old Windows installations
- **Disable Windows Telemetry** - Turn off all data collection services
- **Registry Optimization** - Safe registry tweaks with automatic backup
- **ALL-IN-ONE Complete Optimization** - Runs all optimizations automatically

## 🎯 Usage Instructions

### Basic Usage:
1. **Right-click** on `UltimateSystemTool.cmd`
2. Select **"Run as Administrator"** (Required!)
3. Choose desired function from the menu (0-25)
4. Follow on-screen prompts

### Requirements:
- Windows 7, 8, 10, or 11
- Administrator privileges
- Recommended: Close all applications before running major cleanups

## 📝 Post-Cleanup Recommendations

After running cleanup/optimization:

1. **Restart your computer** for changes to take effect
2. **Run defragmentation** (for HDD only):
   ```cmd
   defrag C: /O
   ```
3. **Check disk health**:
   ```cmd
   chkdsk C: /F
   ```
4. **Scan for viruses** with Windows Defender
5. **Update Windows** to latest version

## ⚙️ What Gets Cleaned/Optimized

### Directories Cleaned:
#### User Level:
- `%TEMP%` - User temporary files
- `%LOCALAPPDATA%\Temp` - Local temp files
- `%APPDATA%\...\Cache` - Application caches
- Recent items, thumbnails, icon cache

#### System Level:
- `C:\Windows\Temp` - System temporary files
- `C:\Windows\Prefetch` - Prefetch data
- `C:\Windows\SoftwareDistribution` - Windows Update cache
- `C:\Windows\Logs` - System logs
- `C:\ProgramData\Microsoft\Windows\WER` - Error reports
- `C:\Windows.old` - Old Windows installation (optional)

#### Browser Caches:
- Chrome, Firefox, Edge, Opera, Brave
- Cache, Code Cache, GPU Cache, Service Worker cache

#### Application Caches:
- Discord, Microsoft Teams, Spotify, Slack, Zoom
- Visual Studio Code, Steam, Adobe applications

### Optimizations Applied:
- Services optimization (disables unnecessary services)
- Visual effects tuning
- Power plan optimization
- Network stack optimization
- Registry tweaks for faster performance
- Startup delay removal
- Telemetry disabling (optional)

## ❓ Troubleshooting

### "Access Denied" errors:
- Make sure to run as Administrator
- Temporarily disable antivirus
- Close all running applications

### Script doesn't work:
- Check Windows version compatibility
- Ensure PowerShell is enabled
- Run Windows Update first

### Slow performance:
- Some operations (like DISM, SFC) can take 30+ minutes
- Don't interrupt the process
- Check if disk is healthy
