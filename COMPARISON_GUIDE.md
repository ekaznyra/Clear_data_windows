## ?? HOW TO RUN / CACH CHAY

### CMD VERSION:
```cmd
1. Right-click UltimateSystemTool.cmd
2. Select "Run as Administrator"
3. Done! / Xong!
```

### POWERSHELL VERSION:
```powershell
1. Right-click UltimateSystemTool.ps1
2. Select "Run with PowerShell"
   (Auto-elevates to Admin)
3. If blocked, run first:
   Set-ExecutionPolicy Bypass -Scope Process
4. Done! / Xong!
```
