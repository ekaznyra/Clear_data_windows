🔧 HOẶC FIX BẰNG TAY (NẾU KHÔNG MUỐN DÙNG UNBLOCK-AND-RUN.ps1):
────────────────────────────────────────────────────────────────────────────

CÁCH A: Dùng Properties (Đơn giản nhất)
────────────────────────────────────────
1. Right-click file UltimateSystemTool.ps1
2. Chọn Properties
3. Ở dưới cùng tab "General", tìm chữ "Security:"
4. Tick vào ô "Unblock" bên cạnh
5. Click OK
6. Chạy lại file bình thường

CÁCH B: Dùng PowerShell Command
─────────────────────────────────
Mở PowerShell as Administrator, paste lệnh này:

Unblock-File -Path "C:\Users\nguye\Downloads\UltimateSystemTool.ps1"

Sau đó chạy:

cd "C:\Users\nguye\Downloads"
.\UltimateSystemTool.ps1
