  HOW TO RUN ON WINDOWS
═══════════════════════════════════════════════════════════════════

METHOD 1: Direct Run (Recommended)
----------------------------------
1. Copy "UltimateSystemTool.cmd" to your Windows PC
2. RIGHT-CLICK on the file
3. Select "Run as Administrator" ⚠️ IMPORTANT!
4. Choose function from menu (0-25)
5. Press Enter

METHOD 2: Test Menu First
--------------------------
1. Copy "TEST_MENU.cmd" to your Windows PC
2. Double-click to test the menu display
3. Verify all text shows correctly in English
4. Then run UltimateSystemTool.cmd as Administrator
