### ?? Selling Your PC

**What to share:**
1. Run [3] Hardware Details

3. Run [4] Storage Info
4. Show disk sizes and health
5. Or use [6] Export Full Report and share file

### ?? Tech Support Call

**What they'll ask:**
1. Run [2] Detailed Report
2. Read OS Version, Build number
3. Check Total RAM
4. Verify Windows Updates installed

**Or:**
- Use [6] Export Full Report
- Email file to support team

---

### ?? Before Upgrading

**Check compatibility:**
1. [3] Hardware Details ? Check motherboard model
2. [3] Hardware Details ? Check RAM speed and type
3. [4] Storage Info ? Check available slots/space

**Example:**
- Want to add RAM? Check Bank Labels to see free slots
- Want to add SSD? Check interface types available
- Want to upgrade GPU? Check current GPU and PSU

---

### ?? Network Issues

**Troubleshooting steps:**
1. Run [5] Network Information
2. Check Connectivity Test results:
   - ? Internet: NOT CONNECTED ? Check router/modem
   - ? Internet: CONNECTED but ? DNS: FAILED ? DNS issue
3. Check IP Configuration ? Look for 169.254.x.x (bad!)
4. Check Active Connections ? Too many? Malware?

---

### ?? Daily Quick Check

**Every morning routine:**
1. Run [1] Quick Overview
2. Check:
   - Available RAM (should have 30%+ free)
   - Disk space (all drives <80%)
   - System uptime (restart weekly)

**Takes: 5 seconds!**

## Troubleshooting

### "Access Denied" when running

**Solution:**
- Right-click WindowsToolkit.bat
- Select "Run as administrator"
- Click "Yes" on UAC prompt

---

### Some hardware info missing

**Causes:**
- Laptop manufacturers hide some info
- Older hardware (pre-2010)
- Virtual machines

**Solution:**
- Run [2] Detailed Report instead
- Use manufacturer's tools for specific info

---

### Export takes too long

**Normal:**
- 30-60 seconds for complete report
- Gathering data from hardware takes time

**If >2 minutes:**
- Close other programs
- Disable antivirus temporarily
- Restart computer and try again

---

### Network test shows "NOT CONNECTED" but I have internet

**Possible causes:**
- Firewall blocking ping
- Router configured to ignore ping
- VPN active

**Check:**
- Can you browse websites? ? Internet works
- Tool tests 8.8.8.8 (Google DNS)
- Some networks block ICMP ping

## FAQ

**Q: How often should I check system info?**
A: Quick Overview daily, Full Report monthly or before major changes.

**Q: Can I share the exported report?**
A: Yes! Remove sensitive info first (computer name, user name, serial numbers).

**Q: Why does storage show less than advertised?**
A: Manufacturers use 1000 MB = 1 GB. Windows uses 1024 MB = 1 GB. Normal!

**Q: What's the difference between Physical and Available RAM?**
A: Physical = Total installed. Available = Currently free to use.

**Q: My GPU shows 0 MB RAM?**
A: Integrated GPUs share system RAM. Tool may not detect VRAM. Normal for Intel/AMD integrated.

**Q: Network adapter shows 1000000000 speed?**
A: That's 1 Gbps (1000 Mbps). Tool shows bits per second.

**Q: Can I run this on Windows 7?**
A: Yes! Works on Windows 7, 8, 10, and 11.

**Q: Does this send any data?**
A: NO! Everything runs locally. No internet required (except connectivity test).
