================================================================
  HOW TO RUN - WINDOWS TOOLKIT
================================================================

SIMPLE STEPS:

1. Right-click on "WindowsToolkit.bat"

2. Select "Run as administrator"

3. Click "Yes" when Windows asks for permission

4. Choose option from menu:
   - Press 1 for Cleanup
   - Press 2 for Activation
   - Press 3 for Software Manager
   - Press 0 to Exit

5. Follow on-screen instructions

================================================================

EXAMPLE - CLEAN JUNK FILES:

1. Run WindowsToolkit.bat as administrator
2. Press 1 (Cleanup Tool)
3. Press 2 (Standard Cleanup)
4. Wait for completion
5. Press any key to go back

================================================================

EXAMPLE - ACTIVATE WINDOWS:

1. Run WindowsToolkit.bat as administrator
2. Press 2 (Windows Activation)
3. Press 1 (KMS Activation)
4. Type Y to confirm
5. Wait for activation
6. Check status with option 3

================================================================

TROUBLESHOOTING:

Problem: "Access Denied"
Solution: You must run as ADMINISTRATOR!
          Right-click -> Run as administrator

Problem: Nothing happens
Solution: Make sure you right-clicked and selected 
          "Run as administrator"

Problem: Script closes immediately
Solution: Don't double-click! Right-click and 
          select "Run as administrator"

================================================================

THAT'S IT! VERY SIMPLE!

No PowerShell needed.
No installation needed.
Just run the BAT file as Administrator.
