## [2.0.0-analytical-pro] - 2025-11-03

### 🎉 Major Release - Analytical Pro Edition

#### ✨ Tính năng mới
- **🔍 Smart Scan System**: Tự động quét và chỉ dọn phần mềm có trong máy
- **🧠 AI-Powered Analysis** (Python): Phân tích thông minh với gợi ý ưu tiên
- **💾 Backup & Restore System**: Tự động backup 24h với khả năng undo
- **📊 Advanced Reporting**: HTML reports với Chart.js, beautiful UI
- **🎯 Menu System**: Interactive menu với 6 chế độ hoạt động
- **🧹 Dev Mode**: Dọn code projects (node_modules, __pycache__, etc)
- **📈 Statistics**: Đếm số phần mềm tìm thấy, thư mục dọn, dung lượng
- **🔒 Enhanced Security**: Double admin check + UAC automation

#### 🎨 Cải thiện
- Giao diện menu đẹp hơn với box drawing characters
- Title bar hiển thị trạng thái (Administrator Mode)
- Progress indicators cho từng bước
- Color coding cho priority levels
- Thông báo rõ ràng, dễ hiểu

#### 📦 Files mới
- `cleanup_analyzer.py` - Python analyzer với pandas + rich
- `requirements.txt` - Python dependencies
- `config.json` - Configuration file
- `quick_start.cmd` - Quick launcher
- `README.md` - Full documentation
- `CHANGELOG.md` - Version history

#### 🐛 Bug fixes
- Fix: Admin check bỏ qua UAC trong một số trường hợp
- Fix: Log file encoding UTF-8
- Fix: Path với spaces không xử lý đúng
- Fix: Script crash khi Temp folder đầy
