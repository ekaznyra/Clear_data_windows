===============================================================================
COMMON ERRORS:
===============================================================================

❌ "Access is denied"
   → Not running as Administrator
   → FIX: Right-click → "Run as administrator"

❌ Window closes immediately  
   → Double-clicked instead of run as admin
   → FIX: RIGHT-CLICK → "Run as administrator"

❌ Antivirus blocks it
   → Windows Defender thinks it's malware
   → FIX: Temporarily disable or add to exclusions

❌ Commands not found
   → System PATH is broken
   → FIX: Read TROUBLESHOOTING.txt
