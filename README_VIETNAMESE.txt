===============================================================================
❌ LỖI THƯỜNG GẶP
===============================================================================

❌ "Access is denied"
→ Chưa chạy với quyền Administrator
→ FIX: Chuột phải → "Run as administrator"

❌ Cửa sổ đóng ngay
→ Double-click thay vì run as admin
→ FIX: PHẢI chuột phải → "Run as administrator"

❌ Antivirus chặn
→ Windows Defender nghĩ là virus
→ FIX: Tắt antivirus tạm thời hoặc thêm vào exclusion

❌ Không có quyền
→ Account không phải Administrator
→ FIX: Đăng nhập bằng account Administrator

❌ File bị lỗi
→ File bị corrupt khi download
→ FIX: Download lại hoặc tạo file mới

✅ KIỂM TRA NHANH
===============================================================================

Test xem máy bạn có chạy được batch file không:

1. Mở Notepad
2. Copy paste:

@echo off
echo Test batch file
pause

3. Save as: test.cmd
4. Chuột phải → Run as administrator
5. Nếu thấy "Test batch file" → Máy OK!
